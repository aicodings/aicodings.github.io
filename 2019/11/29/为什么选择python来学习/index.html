为什么选择python来学习

编程之路漫漫，吾将上下而求所。想了很多这个站到底该取什么样的名字，觉得编程小站，以及编程之路都被人选走了，剩下的就觉得这个名字好听一点。小路，一条狭小的可以容得下自己走的路。这个站主要是分享一些编程方面的知识，主要是python.

为什么有PHP以及C等等语言，而偏偏选择了python，最近看了一个榜单，python的受欢迎程度已经位列榜首了，相信在以后的日子里可能还会更多，第一，学起来相对的简单，高级语言嘛，越接近人的思维习惯，都比较好理解。第二个就是能够和网站结合起来，这个应该是学习它的主要原因了，因为如果真的要学习编程，没有精通那么两到三门的话你怎么好意思说自己会编程呢。第三个原因是一种趋势吧，现在人工智能的兴起，越来越多的程序员开始学习这样一种语言。
